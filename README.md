# GPT4-API-Arabic-Classifier

This repository demonstrates how to use the **OpenAI GPT API** to generate responses in Arabic and classify text into predefined intents. The project uses Python and the official OpenAI library, showing examples of both response generation and intent classification.

## Features

- Generate text responses in **Arabic** from GPT models
- Classify input text into predefined categories:
  - `greetings`
  - `hotel_reservation`
  - `flight_reservation`
  - `education`
  - `none`
- Examples provided for both response generation and classification
- Easy integration with Python scripts or Jupyter notebooks
